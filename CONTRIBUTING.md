# 贡献指南

欢迎各位同学为本项目提交代码优化、文案扩充、界面改进等贡献！

## 1. 开发环境准备
1. 将仓库Fork至个人GitHub账号；
2. 克隆本地：`git clone 仓库地址`；
3. 直接打开 index.html 即可本地运行，无需额外构建工具。

## 2. 分支规范
- 新增功能：`feat/xxx`
- Bug修复：`fix/xxx`
- 文案更新：`docs/text`
- 界面调整：`ui/layout`

## 3. 提交规范
提交描述格式：`类型: 简短说明`
示例：
- feat: 新增考研文案分类
- fix: 修复文字描边错乱问题
- docs: 更新README使用说明

## 4. PR提交流程
1. 本地修改完成后推送至个人Fork仓库；
2. 向主仓库 main 分支发起 Pull Request；
3. 等待团队审核，通过后合并代码。

## 5. 贡献内容范围
1. 扩充12大类校园文案素材；
2. 优化画布渲染、文字样式逻辑；
3. 适配移动端布局，修复兼容性Bug；
4. 完善项目文档、使用教程。

## 6. 开源规范约束
所有提交代码遵循本仓库 MIT LICENSE 协议，贡献代码即代表同意开源共享。
