# 🧠 精神状态生成器

> 大学生专属表情包&文案一键生成工具，纯前端离线运行，无需后端、不用数据库，浏览器本地即可使用。

## ✨ 功能特点

- 📝 **丰富文案库**：12大分类，150+条精选文案，覆盖大学生活各个场景
- 🎨 **个性化编辑**：自定义文字颜色、字号、描边、对齐方式
- 🖼️ **多样背景**：纯色背景切换、12种表情包底图可选
- 📤 **一键导出**：高清PNG图片导出，支持直接分享
- 💾 **本地存储**：文案收藏、用户配置自动保存
- 🌓 **暗黑模式**：支持亮色/暗色主题切换
- 📱 **响应式设计**：完美适配手机和电脑端

## 🚀 快速开始

### 方式一：直接打开

1. 下载项目到本地
2. 双击打开 `Mental-State-Generator.html`
3. 浏览器中即可使用

### 方式二：本地服务器（推荐）

```bash
# 使用 Python 启动简单服务器
python -m http.server 8000

# 使用 Node.js
npx serve .

# 使用 PHP
php -S localhost:8000
```

然后访问 `http://localhost:8000/Mental-State-Generator.html`

## 🎮 使用教程

### 1. 选择文案
- 点击左侧分类按钮，快速切换文案分类
- 点击「🎲 随机文案」按钮，随机生成一条文案
- 也可以在输入框中手动输入自定义文案

### 2. 调整样式
- **画布尺寸**：支持方形(600×600)和竖图(600×900)
- **背景颜色**：点击颜色选择器自定义背景色
- **表情包底图**：点击右侧表情包按钮切换底图
- **字号**：拖动滑块调整文字大小
- **文字颜色**：点击颜色选择器自定义文字颜色
- **描边粗细**：拖动滑块调整文字描边

### 3. 添加装饰
- 点击贴纸按钮添加/移除装饰贴纸
- 支持爱心、星星、彩虹、太阳等8种贴纸

### 4. 导出分享
- 点击「⬇️ 导出PNG」按钮，下载高清图片
- 点击「📋 复制文案」按钮，复制文案到剪贴板

### 5. 收藏管理
- 点击「⭐ 收藏当前文案」按钮收藏喜欢的文案
- 点击收藏列表中的文案可快速填充到输入框

## 🛠️ 技术栈

- **HTML5**：页面结构
- **CSS3**：样式设计
- **JavaScript (ES6+)**：核心逻辑
- **Canvas API**：画布绘图和图片导出
- **Tailwind CSS**：快速样式开发（CDN引入）
- **LocalStorage**：本地数据存储
