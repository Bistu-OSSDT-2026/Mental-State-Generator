# 🎉 页面交互

BISTU专属表情包&文案页面交互

## ✨ 功能特点
- **样式自定义**：支持自定义文字颜色、背景颜色、字体样式
- **BISTU水印**：可开关的校徽水印，支持四个位置切换
- **主题切换**：支持深色/浅色主题
- **一键导出**：支持复制文案和下载图片（Canvas导出）
- **历史记录**：保存最近5条生成记录
- **纯前端运行**：无需后端，所有数据仅存本地浏览器

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计
- **Tailwind CSS** - 快速布局
- **JavaScript** - 交互逻辑
- **Canvas API** - 图片导出
- **Font Awesome** - 图标库

## 🚀 快速开始

### 方式一：直接打开

直接在浏览器中打开 `index.html` 文件即可使用，无需任何配置。

### 方式二：本地服务器

```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用start_server.py
python start_server.py
```

然后访问 `http://localhost:8000`

## 🎯 使用说明

1. **选择分类**：点击顶部导航栏选择文案分类
2. **输入关键词**：在左侧输入区域填写关键词
3. **选择情绪**：点击情绪标签筛选匹配的表情
4. **选择风格**：选择文案风格（幽默搞笑、文艺诗意等）
5. **自定义样式**：调整文字颜色、背景颜色、字体等
6. **一键生成**：点击生成按钮获取表情包
7. **导出使用**：复制文案或下载图片

## 🔒 隐私安全

- 纯前端离线运行，无需登录
- 所有数据仅存储在本地浏览器
- 不会上传任何用户数据到服务器
